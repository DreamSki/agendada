## 💡 修复建议优先级

### P0 - 立即修复

1. **`snapshot()` 主线程阻塞**
   - 文件: ObservableLibraryStore.swift:703
   - 修复: 移到 Task 内部执行
   - 预计收益: 消除保存时卡顿

2. **`rowPositions` 无限增长**
   - 文件: RelatedPanelContentView.swift
   - 修复: 添加清理逻辑，限制最大大小
   - 预计收益: 防止长时间运行后变慢

### P1 - 尽快修复

1. **`publishChange()` 级联刷新**
   - 修复: 使用精细化的观察者模式
   - 预计收益: 减少 50% 的视图刷新

2. **`daySchedules` 持续增长**
   - 修复: 清理旧数据（>90 天）
   - 预计收益: 限制内存增长

### P2 - 后续优化

1. **`filteredNotes()` 缓存优化**
   - 修复: 分离缓存依赖
   - 预计收益: 提升缓存命中率到 >95%
